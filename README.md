# booking-learning

- use node version 16.14.2
- linux
	 + curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
	 + nvm install 16.14.2
	 + nvm use 16.14.2	
	 + node -v    // check version node current
  
Steps for install:
- `cp env.example .env`
- `npm i`
  
Steps for build:
- `npm run build`

Steps for start:
- `npm start`
